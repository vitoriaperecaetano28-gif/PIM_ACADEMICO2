

# Sistema Acadêmico - PIM
O objetivo do PIM é desenvolver um sistema acadêmico simples que permita gerenciar dados de alunos, usuários e notas. O projeto integra conteúdos das disciplinas do semestre, utilizando a linguagem C para criar arquivos CSV e Python para ler e exibir essas informações em uma interface gráfica básica. O sistema demonstra a aplicação prática da programação, modelagem e organização de um pequeno software, atendendo às necessidades de um ambiente educacional de forma simples e funcional.
